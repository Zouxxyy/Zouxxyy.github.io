生成文章

  1. 使用命令生成初始文章，文件名之间有空格的话会自动加-中横线

hexo n "hello hexo" #双引号中填写要生成的文章名

  1. 在hexo/source/_posts中编辑生成出来的hello-hexo.md文件
  1. 本地预览文章内容，登录localhost:4000查看效果

hexo s

  1. 内容发布到public文件夹数中，然后手动复制同步GitHub

hexo clean && hexo g && hexo d
